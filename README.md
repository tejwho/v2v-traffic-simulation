# 🚗 V2V Smart Traffic Simulation

This project demonstrates **Vehicle-to-Vehicle (V2V) communication** using a traffic simulation built with **Python and Pygame**.

## Features

- Multiple vehicles moving on a road network
- Vehicle ID tracking
- Distance-based V2V communication
- Fog condition simulation
- Real-time monitoring dashboard
- Visual communication lines between vehicles

## Technologies

- Python
- Pygame
- Mathematical modeling for vehicle movement

## How to Run

```bash
pip install -r requirements.txt
python main.py
